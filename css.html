THE FUNDAMENTALS OF CSS

CSS is a language for specifying how documents are presented to users — how they are styled, laid out, etc.CSS can be used for very basic document text styling — for example, for changing the color and size of headings and links. It can be used to create a layout — for example, turning a single column of text into a layout with a main content area and a sidebar for related information.
