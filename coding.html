    • HTML vs. CSS

    Html (HyperText Markup Language) is the standard coding language used to structure websites, while Css (Cascading Style Sheets) modifies the appearance of the pages. Essentially, it describes the display of html.


    JAVASCRIPT

    Javacript is basically the more advanced version of CSS. It is more dynamic and allows for more specifications. However, it is also more difficult than CSS for beginners.
